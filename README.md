Форма регистрации студентов - Ручное тестирование 

Описание:
В проекте выполнено ручное тестирование регистрационной формы для студентов и валидация данных с помощью SQL запросов.
Цель: 
Отработка навыков составления QA - артефактов и применения техник тест - дизайна. А также показать понимание связи между UI-действиями пользователя и сохранением данных в базе данных, а также навыки работы с SQL.

Условия:
Возраст студентов от 14 до 70 лет
Обязательные поля: First Name, Last Name, Mobile Number, Gender, Date of Birth, State and City.
Строка Mobile Number принимает только 10 символов и только цифры.
Email не обязателен, но должен быть валидным 
Максимальный размер файла 5МБ

Область тестирования:
Функциональное тестирование
Негативное тестирование 
Валидация данных

Структура данных:
Таблица students
Поля:
first_name
last_name
email
phone
birth_date
gender
state
city
